devDependencies - обяз зависимости для разработки, загружаются с флагом --save-dev
dependencies - конкретно для этой, загружаются с флагом --save

Линтеры я отключил, так как мешают быстрой разработке, а красота кода не так уж и важна,
включить их обратно: в tasks (styles, scripts, pug2html) раскоментировать строку с линтерами + включить плагины линтеров а VS Code
