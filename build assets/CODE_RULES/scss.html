Что делать со стилями

Структура файлов стилей

Любой элемент каскадно располагается по слежующей логике:

  • Эл. встречается 1 раз - добавляется прямо в структуру
  • Эл. встречается 2 и более раз внутри страницы - выноситься в локальный модуль и импортируется
  • Эл. встречается на других страницах, но не на всех - выноситься в модуль local_modules, импортируется отдельно
  • Эл. встречается на всех страницах - выноситься в модуль local_modules, импортируется внутри local_modules, и уже оттуда импортируется вся сборка модулей.

core/base - не изменяется вручную, только через переменные

Кодирование

Каждое определение стилей в блоке делится на 4 части, разделенные пустой строкой:
  • что за блок и его размерности и внутренние отступы (все то что вилется на размер и позиции внутри блока)
  • стили блока, визуальное отображение, которое не влияет на позиционирование
  • все что связано с анимациями и трансформациями для блока, те стили, которые анимируются
  • все что относситься к шрифтам, так как они наследуются их стоит выделить в отдельный блок
  • позиционирование элемента внутри родителя
.body {
  display: flex;
  align-items: center;
  justify-content: space-between;
  width: 18.75rem;
  height: 29.25rem;
  padding: 1.1875rem 1.5rem 0;
  border: 0.125rem solid;

  background: #FFFFFF;
  box-shadow: 0px 9px 60px rgba(0, 0, 0, 0.12);

  transition: all 0.3s ease;
  transform: rotate(90deg);
  border-color: transparent;
  
  font-size: 22px;
  font-weight: normal;
  line-height: 1.5em;
  color: white;

  margin-top: 4rem;
  margin-bottom: 2.2rem;
  position: absolute;
  left: 0;
  bottom: 5rem;
}
